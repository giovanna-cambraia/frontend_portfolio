Projetos

Projeto 1

Este projeto moderno combina um estilo CSS limpo com interatividade aprimorada por JavaScript, apresenta animações acionadas por rolagem, tipografia em gradiente, e mídia incorporada. As seções são animadas dinamicamente, utilizando JavaScript.

Projeto 2

Este projeto apresenta uma animação 3D feita inteiramente com CSS, apresenta transformações avançadas, controle de perspectiva e transições suaves, e reage à interação, oferecendo uma experiência dinâmica sem depender de bibliotecas externas.

Projeto 3

Este código importa e utiliza a biblioteca Three.js para carregar um modelo 3D animado de uma abelha, que se move e rotaciona conforme o usuário rola a página. A câmera e a iluminação são configuradas para destacar o modelo. Também é usado gsap para animações suaves.

Projeto 4

Este código adiciona efeitos visuais de paralaxe baseados no movimento do mouse e animações para abrir/fechar uma página de login. Elementos com a classe .parallax se movem suavemente conforme o cursor se desloca. A página de login desaparece com uma transição.

Projeto 5

Aplicação interativa em React que integra modelos 3D, estilização moderna com Tailwind CSS e animações responsivas. O projeto exibe objetos 3D animados que reagem ao scroll e à posição do usuário, oferecendo uma experiência visual imersiva e fluida.

Projeto 6

Site de arquitetura bilíngue (português/inglês) feito em React com CSS. Apresenta projetos, equipe e serviços com layout responsivo. Usa API REST (Postman) para envio de mensagens via formulário de contato. Ideal para portfólios profissionais acessíveis globalmente.
