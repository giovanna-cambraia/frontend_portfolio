Services

Banco de Dados

Possuo conhecimentos em bancos de dados relacionais, com ênfase no uso de SQL para criação de tabelas, consultas, inserções, atualizações e exclusões de dados. Sei modelar estruturas de dados de forma lógica e eficiente, garantindo integridade e desempenho. Sou capaz de integrar bancos de dados com aplicações backend, utilizando linguagens como Java e JavaScript.

Frontend Development

Possuo experiência com CSS, sendo capaz de estilizar páginas web com layouts personalizados, responsividade e boas práticas de design visual. Com o uso de Tailwind CSS, acelero o desenvolvimento visual através de classes utilitárias, criando interfaces modernas e consistentes com mais agilidade e organização no código. Eu também domino React, uma das principais bibliotecas JavaScript para construção de interfaces dinâmicas.

Backend Development

Possuo conhecimentos em Java e JavaScript desenvolvimento backend, com capacidade para criar APIs, lidar com lógica de negócio, manipulação de dados e integração com bancos de dados. Tenho domínio em ambientes como Node.js, possibilitando a construção de servidores web, rotas, middlewares e manipulação de requisições HTTP. Também estou familiarizada com o uso de Git Bash e versionamento de código.
